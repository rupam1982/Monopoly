PROJECT COMPLETED: Full-featured Flask web application for property management

================================================================================
                              INSTALLATION
================================================================================

1. Navigate to WebInterface:
   cd WebInterface

2. Install Flask:
   pip install -r requirements.txt

3. Start server:
   python run.py

4. Open browser:
   http://127.0.0.1:5000

================================================================================
                           API ENDPOINTS
================================================================================

Route              Method    Purpose
─────────────────────────────────────────────────────────────────────────────
/                  GET       Serve web interface
/api/areas         GET       Get all areas from Asset_database.json
/api/assets/<>     GET       Get properties in an area
/api/players       GET       Get all existing players
/api/assign        POST      Assign property to player (calls CLI)
/api/database      GET       View current Player_database.json
/api/player-assets GET       Get properties owned by player

================================================================================
                         EXAMPLE WORKFLOW
================================================================================

1. Open http://127.0.0.1:5000

2. Select Player:
   → Choose "Player 1" or "+ Add New Player"

3. Select Area:
   → Dropdown now enabled, shows all areas from Asset_database.json

4. Select Property:
   → Shows only properties in selected area

5. Select Houses:
   → Choose 0-4 houses

6. Click "Assign Property":
   → Calls player_asset_assignment.py in background
   → Updates Player_database.json
   → Shows success/error message
   → Refreshes data

                        REQUIREMENTS
================================================================================

Browser: Any modern browser (Chrome, Firefox, Safari, Edge)

  - DatabasePackage/ (Python module)

================================================================================
                        TROUBLESHOOTING
================================================================================

Issue: "Asset database not found"
→ Generate it: python -c "from DatabasePackage import excel_to_assets_json; excel_to_assets_json('Asset_database.xlsx')"

Issue: "Port 5000 already in use"
→ Edit WebInterface/run.py, change port=5000 to port=5001

Issue: Dropdowns show no options
→ Check Asset_database.json exists in parent directory (not WebInterface)

Issue: Flask not found
→ Run: pip install -r requirements.txt
